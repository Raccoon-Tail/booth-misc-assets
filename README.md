# Booth Asset Support / Issue Tracker

このリポジトリは、Boothで配布しているアセット向けの **サポート用Issue置き場** です。  
ソースコードやデータ本体はここには含まれていません。

ここでまとめているアセットは、**基本的に無保証での提供** となります。  
不具合報告をいただいても、修正やアップデートが行われることを保証するものではありません。

## 使い方 / How to use

Boothで購入・ダウンロードしたアセットについて、次のような内容があれば Issue を立ててください。

- 不具合報告（バグ、想定外の挙動など）
- 機能追加・改善の要望
- 使い方についての質問
- ライセンスや利用範囲の確認

新しい Issue を作成する際は、テンプレートの項目を埋めていただけると助かります。

## Issue を立てる前に

- Boothの商品ページに記載されている説明や注意事項を一度ご確認ください。
- すでに同じ内容の Issue が立っていないか、検索してみてください。

## 報告時に書いてほしい情報（例）

- どのアセットか（商品名やURL）
- 利用環境（Unity / VRChatのバージョンなど）
- 再現手順
- 期待していた動作
- 実際に起きた動作（エラーメッセージなど）

できる範囲で構いませんが、詳しく書いていただけるほど対応しやすくなります。

## 注意事項

- 公開したくない情報（個人情報・ワールドのクローズド情報など）は書かないでください。
- ライセンス上公開できないスクリーンショットやデータは添付しないでください。
- 対応の優先度やタイミングは保証できません。
